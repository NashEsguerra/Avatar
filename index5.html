Autonomous Mode

Generate ROS Messages:

  • In Unity, look for “Robotics” in the upper left corner of the screen.
  • Select “Generate ROS messages…”.
  • Replace the ROS message path with the correct one by pressing the ‘Browse’ button then type ‘C:/pick-and-place/Unity-Robotics-Hub/tutorials/pick_and_place/ROS’.

Rebuild the Project:

  • moveit_msgs:
  • - msg
  • - RobotTrajectory.msg
  • niryo_moveit:
  • - msg
  • - NiryoMoveitJoints.msg
  • - NiryoTrajectory.msg
  • - srv

Set Up the Publisher in the Scene:

  • In the Scene tab, under the Hierarchy, look for an object named “Publisher”.
  • Select “Publisher”. The “Inspector” tab will appear on the right side of the screen.
  • Click “Add Component” at the bottom of the Inspector tab.
  • Add the component “Script > Trajectory Planner”.

Assign Elements in Trajectory Planner:

  • Drag and drop elements from the Hierarchy to the newly added components in the Inspector:
    • Drag “niyro_one” from the Hierarchy to the “Niyro One” field in the Trajectory Planner.
    • Drag “Target” from the Hierarchy to the “Target” field in the Trajectory Planner.
    • Drag “TargetPlacement” from the Hierarchy to the “Target Placement” field in the Trajectory Planner.

Update Button Event:

  • In the Hierarchy tab, find and select “Canvas”.
  • Within the Canvas, select the button you wish to configure.
  • In the Inspector tab, scroll down to the OnClick() event section.
  • Replace the “SourceDestinationPublisher.Publish” event with “Trajectory Planner.PublishJoints”:
    • Click the existing “SourceDestinationPublisher.Publish” to highlight it.
    • Click the dropdown menu and select “Trajectory Planner”.
    • From the next dropdown, select “PublishJoints”.

Run the Autonomous Mode:

roslaunch niyro_moveit part_3.launch

Note:
Ctrl + C - shortcut keys to stop the previous script in Windows PowerShell.
